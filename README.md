# teamProfileGenerator

This will generate team profile HTML page using node.js. and the video includes generating HTML file and testing.



## Technologies

- HTML
- CSS
- JavaScript
- Node.js

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
